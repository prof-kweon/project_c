공통 제약 사항
    크로스 플랫폼을 지원하는 개발 도구를 사용한다.
    이 과정에 사용하는 C 언어 문법은 ANSI C 표준을 따른다.
    ANSI C 표준을 준수하는 C 언어 컴파일러로 gcc 9.x를 사용한다.
    C언어 표준 라이브러리 중 stdio.h, stdlib, math.h, time.h만 사용한다.
    ANSI C 표준 코딩 스타일을 따른다.


코딩 가이드
    프로젝트 폴더의 이름은 Magrathea로 한다.
    소스 코드에 문제 해결과 관련 없는 코드는 포함하지 않아야 한다.
    각 문제 별로 지정한 이름의 코드 파일에 작성해야 한다.
    제출하는 코드는 반드시 주석을 포함해야 한다.
    포인터 변수를 사용하지 않는다.
    이 과정의 메뉴는 “1. 체력과 지식”이다.
    상위 메뉴와 하위 메뉴간의 이동이 가능해야 한다.
    프로그램 종료는 주 메뉴에서 수행한다.
    어떤 메뉴 경로에서 든 주 메뉴로 갈 수 있어야 한다.
    main 함수의 내용은 최소화 되어야 하며 문제는 함수를 구현하여 해결해야 한다.
    문제를 해결하는데 필요한 함수의 필요한 반환 형식과 매개변수는 별도로 언급하지 않은 경우는 자유롭게 구현한다 .
    이 과정은 과제 1의 범위를 포함해 함수와 다차원 배열까지 사용할 수 있다.
